
# VotoLyze
Backend do projeto VotoLyze - Disciplina de Engenharia de Software 2 - IFSP Campus São João da Boa Vista

# Requisitos
Java 17, Maven

# Iniciar o servidor
Abrir a pasta do projeto no terminal e executar os comandos: 

```mvn clean build```

```mvn spring-boot:run```
